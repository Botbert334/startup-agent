# GitHub Pages setup

1) Go to this repo on GitHub → **Settings** → **Pages**.
2) Under **Build and deployment**:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/docs**
3) Save.

Your site will publish at:
https://botbert334.github.io/startup-agent/

(If you want a different repo name / URL, rename the repo or use a custom domain.)
